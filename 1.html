HTML이란 무엇인가?

Hypertext Markup Language(HTML) is the is the standaed markup language for creating web pages and web applications. HTML also provides a way to create structured documents with links, citations, and other items, as well as representing structural meaning for text such as headings, paragraphs, lists, etc. And it can be used to create interactive forms by embedding images and objects. HTML is written in the form of HTML elements consisting of "tags" enclosed in angle brackets within the content of a web page .

HTML may contain or load scripts such as JavaScript , which affect the behavior of HTML processing devices, such as web browsers , andCSS , which define the appearance and placement of text and other items . W3C , co-director of HTML and CSS standardsrecommends the use of CSS for clear and expressive markup.
